## 📝 Cómo Actualizar la Aplicación

### Opción 1: Automática (Recomendado)

Cada vez que hagas un `push` a GitHub, Netlify se actualizará automáticamente:

```bash
git add .
git commit -m "Cambios en la app"
git push origin main
```

Netlify detectará el push y reconstruirá la aplicación automáticamente (~2-3 minutos).

### Opción 2: Manual

```bash
# Desde el directorio del proyecto
netlify deploy --prod --dir=build
```
